# Generated Font Atlases

This directory contains generated font atlases and metrics.

Generated by: `cargo run -p xtask -- fonts`

Files:

- `*.msdf.png` - MSDF atlas textures
- `*.metrics.json` - MSDF glyph metrics
- `*.bitmap.png` - Bitmap atlas textures
- `*.bitmap.json` - Bitmap glyph metrics

**These files are checked into the repository** to ensure:

- Fast builds (no generation during normal compilation)
- Reproducibility (deterministic outputs)
- CI stability (no external tool dependencies during build)
